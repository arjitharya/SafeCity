CityOS prototype

Files:
- index.html: dashboard structure
- styles.css: custom CityOS styling, responsive layout, dark mode
- app.js: sidebar menu, live clock, alerts, dashboard filtering, map markers, simulated telemetry

How to use:
1. Keep all three files in the same folder.
2. Open index.html in a browser.
3. Use the left sidebar to filter modules.
4. Use the search box to filter visible content.
5. Click map markers to change district data.
6. Click the moon/sun button to toggle dark mode.

This is front-end only. To make it a real CityOS, connect app.js to an API, Firebase, Supabase, or a Node backend.
